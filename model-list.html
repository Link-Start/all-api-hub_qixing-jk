模型列表与价格比对

一眼看清所有中转站的模型底细。支持跨站比对价格、自动锁定最优分组、批量验证可用性，助你做出最实惠的选择。

功能概览

  • 💰 实时价格比对：自动根据计费倍率、分组倍率和汇率，计算各模型在不同站点的实际单价。
  • 🎯 最优路径推荐：在多账号视图下，系统会自动标出该模型的“全站最低价”或当前站点的“最优分组”。
  • 🔍 多维深度过滤：支持按站点来源、API 凭据、提供商（OpenAI, Anthropic 等）、计费方式和标签进行组合筛选。
  • ✅ 批量可用性验证：支持模型连通性测试、Token 兼容性检测以及 CLI 代理可用性验证。
  • 📊 计费透明化：清晰区分“按量计费（Tokens）”与“按次计费（Per Call）”，并提供官方倍率对照。

进入方式

  1. 在扩展弹窗中点击左下角的 “模型列表” 图标。
  2. 或在 设置 → 模型管理 中进入。

核心操作指南

1. 选择数据源

在页面顶部的选择器中，你可以切换不同的数据源：

  • 全部账号：聚合你已添加的所有站点账号，开启跨站比对。
  • 特定账号：查看指定站点的完整模型目录。
  • API 凭据：查看 API 凭据库中已保存 Key 支持的模型（无需创建站点账号）。

2. 智能价格比对

模型卡片会展示该模型的详细计费信息：

  • 计费倍率：展示站点的模型倍率（如 1.5）和补全倍率（如 2）。
  • 分组倍率：显示你当前所属分组（如 default, VIP）的额外倍率。
  • 实际单价：系统会自动折算为 每 1M Tokens (USD/CNY) 或 每次调用 的价格。
  • 最低价标识：在“全部账号”视图下，价格最优的站点会带有绿色标签。

3. 分组切换与模拟

如果一个站点支持多个用户分组（如 default, vip, svip），你可以在筛选栏中切换分组，实时预览不同等级下的价格变化。

💡 技巧：系统默认会为你选择该模型最实惠的可用分组。

4. 模型验证

点击模型卡片右侧的验证图标：

  • 验证模型：发送一个轻量级请求，确认该模型当前是否真实可用。
  • 验证 CLI 兼容性：测试该模型是否支持流式输出，以及是否能被命令行工具正常调用。
  • 批量验证：点击工具栏的 “批量验证” 为当前过滤出的所有模型排队执行测试。你也可以通过复选框勾选特定的模型，仅对选中的部分进行批量验证。

列表控制项

配置项说明
搜索框支持模型 ID（如 gpt-4o）或站点名称模糊搜索。
计费模式可筛选“按量计费”或“按次计费”的模型。
提供商筛选快速锁定 OpenAI、Anthropic、Google、Meta 等厂商的模型。
排序方式支持按名称、价格（从低到高）、账号数量进行排序。
显示设置可控制是否显示人民币价格、是否显示倍率列、是否显示服务端类型。

常见问题

问题解决方案
为什么有的模型显示价格为 0？可能是该站点将此模型设为免费，或者站点未提供有效的计费数据，此时会回退到默认目录。
跨站比对时，汇率是怎么计算的？插件内置了 1 USD = 7.3 CNY 的固定汇率（参考 New API 约定），主要用于统一比对基准。
“最优分组”是什么意思？当模型在多个分组（如 default 和 vip）中都可用时，插件会自动选择倍率最低的分组来计算价格。
为什么验证结果显示“未知”？请检查你的 API Key 是否有权限调用该模型，或者站点当前是否触发了频率限制。
